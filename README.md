# Homebrew Tap for PrayerTimes

macOS menu bar app for Islamic prayer times with Hijri calendar.

## Installation

```bash
brew tap abd3lraouf/prayertimes
brew install --cask prayertimes
```

## Update

```bash
brew upgrade --cask prayertimes
```

## Uninstall

```bash
brew uninstall --cask prayertimes
```
